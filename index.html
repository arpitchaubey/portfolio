Money Mentor

Money Mentor is a sophisticated financial advisor landing page featuring an integrated chatbot tailored to handle typical financial inquiries. This project exemplifies seamless integration between front-end web development and the Python Flask framework. The technology stack includes HTML and CSS for the front-end, while Flask serves as the backend framework. Python libraries such as PyTorch and NLTK enhance functionality. Together, these components enable robust and responsive user interactions, setting a high standard in modern web development for financial services.
